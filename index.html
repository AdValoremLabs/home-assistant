# Home Assistant Kit

Adding value to your life

A plug-and-play smart-home hub built on **Home Assistant OS** and **Raspberry Pi 5 (4 GB)**. Includes a pre-flashed SD card, verified components, and a quick-start card. **Docs & files:** - GitHub repo → - Quickstart (HTML) → `home-assistant/docs/quickstart.html` in the repo - FAQ → ## What’s in the kit - Raspberry Pi 5 (4 GB) with active cooling - 32 GB A2 microSD (pre-flashed HAOS) - 27 W USB-C PSU · Micro-HDMI→HDMI cable · Printed quickstart ## First boot 1. Plug in Ethernet (recommended), insert SD, power on. 2. Wait 5–10 min for first-boot setup. 3. Visit **http://homeassistant.local:8123/** (or the Pi’s IP). > Need Wi-Fi? See the `my-network` example in the repo.
